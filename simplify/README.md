## Introduction

This command parses a list of abstract WCET formula, simplifies them,
and prints the simplified formulas on standard output.

## Dependencies

The following packages are required to compile:

- ocaml

- ocamlbuild

- make

## Compilation

Compile with `make`.

## Execution

Execute with `./simplify`.
